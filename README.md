## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

 - NPM updated
 - Modern web browser
 - Git
### Setup

    git clone https://github.com/indigodavid/react-tutorial.git
    cd react-tutorial/
### Install

    npm install
### Usage

    npm run build
    npm start
### Deployment

    npm run deploy
